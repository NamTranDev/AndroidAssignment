## Introduce

The sample app shopping product from api [https://dummyjson.com/docs] demo for jetpack compose

## More Features:
Migrade to clean architecture 

## Features:
- OnBoarding.
- Product List.
- Search Product.
- Product Detail.
- Bookmark Product.
